## Evidências de Teste

Esta pasta contém as evidências da execução dos testes realizados na API ViaCEP.

As evidências consistem em vídeos com áudio da execução manual dos testes no Postman, demonstrando:
- Status HTTP retornado
- Corpo da resposta da API
- Comportamento esperado para cada cenário testado

Os arquivos estão nomeados de acordo com o identificador do caso de teste correspondente (CT01, CT02, etc.). Devido ao tamanho dos arquivos, o GitHub não realiza a visualização direta,
sendo necessário realizar o download para visualização local.
